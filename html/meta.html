meta要素

文書に関するいろいろな情報を記載する要素です。

文字コード、ホームページの説明文、検索エンジン向けのキーワードなど。

要素自体は必須ではなく、また、閉じタグが不要です（閉じタグが不要な要素を空要素と言います。）

<!DOCTYPE html> <html lang="ja"> 　<head> 　　<meta charaset="UTF-8"> 　　<title>がんばって勉強しろ。</title> 　</head> </html>


文字コードの指定

charset属性で文字コードを指定します。

上記のようにhead要素の開始タグの直後に指定することで文書全体に文字コードが反映されます。

主な文字コード

  • UTF-8:HTML5が推す可変長マルチバイトな文字コード（多言語）
  • EUC-JP:UNIXで使用する文字コード（日本語）
  • Shift-JIS:Windowsで使用する文字コード（日本語）

文字化けしないように注意しましょう( *´艸｀)


サイトの説明

name属性にdescriptionを指定することでホームページの説明文を表します。

検索エンジンの検索結果に表示される文章となるので、SEO的に重要な項目となります。

<!DOCTYPE html> <html lang="ja"> 　<head> 　　<meta charaset="UTF-8"> 　　<meta name="description" content="HTML がんばって勉強しろ。"> 　　<title>がんばって勉強しろ。</title> 　</head> </html>

キーワード

name属性にkeywordを指定することで検索エンジンに引っかかる確率が上がります。

…と言われていましたが、フィッシング的な不正利用が多発したため、今はそうでもないらしいです(´・ω・`)ｼｮﾎﾞﾝ

<!DOCTYPE html> <html lang="ja"> 　<head> 　　<meta charaset="UTF-8"> 　　<meta name="description" content="HTML がんばって勉強しろ。"> 　　<meta name="keyword" content="HTML,CSS,javascript,PHP"> 　　<title>がんばって勉強しろ。</title> 　</head> </html>
