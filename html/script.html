script要素

javascriptをHTML文書内に記述する際に指定する要素です。

ブラウザのレンダリングを邪魔しないように、body要素の最後に記述することが一般的です。

<!DOCTYPE html> <html lang="ja"> 　<head> 　　<meta charaset="UTF-8"> 　　<meta name="description" content="HTML がんばって勉強しろ。"> 　　<meta name="keyword" content="HTML,CSS,javascript,PHP"> 　　<link rel="stylesheet" href="css/style.css" type="text/css"> 　　<link rel="shortcut icon" href="images/app.ico"> 　　<base href="http://yuumin.chu.jp/htmlstudy"> 　　<style type="text/css"> 　　　　h1 { text-align: center;} 　　</style> 　　<title>がんばって勉強しろ。</title> 　</head> 　<body> 　　<script type="text/javascript" src="script.js async"> 　　</script> 　</body> </html>

上記ではtype属性を指定していますが、HTML5ではデフォルトが"text/javascript"のため記載を省略可能となりました。

javascriptの実装を直接タグの中に記述することも可能ですし、src属性を使って外部ファイルを読み込むこともできます。


また、上記のようにasync属性を付けると"script.js"が読み込まれた時点で1度実行されます。

asyncではなく、defer属性を付けるとWebサイト(htmlファイル)全体が読み込まれた時点で1度実行されます。
