noscript要素

javascriptが無効になっている場合に実行される要素です。

一般的に、ブラウザのscript機能を無効にされた際に、警告メッセージを表示する場合に使用されます。

<!DOCTYPE html> <html lang="ja"> 　<head> 　　<meta charaset="UTF-8"> 　　<meta name="description" content="HTML がんばって勉強しろ。"> 　　<meta name="keyword" content="HTML,CSS,javascript,PHP"> 　　<link rel="stylesheet" href="css/style.css" type="text/css"> 　　<link rel="shortcut icon" href="images/app.ico"> 　　<base href="http://yuumin.chu.jp/htmlstudy"> 　　<style type="text/css"> 　　　　h1 { text-align: center;} 　　</style> 　　<title>がんばって勉強しろ。</title> 　</head> 　<body> 　　<noscript> 　　　　ブラウザのscript機能を有効にしてください。 　　</noscript> 　　<script type="text/javascript" src="script.js async"> 　　</script> 　</body> </html>

noscript要素内に記載された処理やメッセージはブラウザのscript機能を無効にしないと実行されません。
