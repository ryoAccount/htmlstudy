animate

要素を移動させたり大きくしたり、アニメーション（動き）を与える際に使用します。

$('.left').click(function() { 　$('.moving').animate( {'marginLeft' : '-=50px'} ); }); $('.right').click(function() { 　$('.moving').animate( {'marginLeft' : '+=50px'} ); });

animate関数の内容は"margin-left"（ボックスの外側の余白を指定するCSS）に対して50px増減させる処理内容になります。

もし、"-="や"+="がなかったら、"margin-left"に50pxを設定する処理になります。


実際の表示

ボタンをクリックすると動きます。
