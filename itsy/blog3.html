しかし、実際の開発の現場ではアプリケーションから外部のサービスを呼び出す際のIFの仕様という意味で使われることが多い。
APIも使いこなせば非常に強力なツールとなります。どんどん使いましょう。

Face

Face はMicrosoftが開発・提供している顔認証APIです。
人が映った写真を2枚並べ、一致度合いを測定してくれます。
一致している可能性が高いほど大きい数値、人違いであるほど小さい数値が出力されます。

Face の凄いところは顔だけでなく、怒りや驚き、喜びといった感情も検出します！
さすがはMicrosoft。preziにPowerPointを罵倒されたけど、技術力は負けていない。

Geo Location API

Geolocation API はユーザの位置情報を扱うAPIです。
このAPIはどこかのイケてるIT企業が開発しているWebサービスではなく、
JavaScriptに標準で搭載されている機能・仕組みとなります。

Geolocation API で取得できるのは緯度・経度だけではありません。
方角・高度・速度も得ることができます。

Geolocation API を使ったサービスで一番に思い付くのは・・GoogleMap！
カーナビやPokemonGoなども、このAPIを利用したサービスと言えます。
